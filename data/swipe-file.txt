# Swipe file - linki do KANALOW (lub pojedynczych filmow) ze short formem.
#
# JAK TO DZIALA:
# 1. Z kazdego kanalu pipeline wyciaga top shorty (YT) lub najnowsze (TT/IG).
# 2. Dla YT: filtr po wyswietleniach (default >= 500 000, zmienialny w .env).
# 3. Dla TT/IG: bierze N najnowszych (TT/IG nie udostepnia view_count w listach).
# 4. Z calej puli losuje 3 codziennie i generuje 3 nowe skrypty.
# 5. Wysyla na Telegram.
#
# WAZNE: TT i IG wymagaja COOKIES w data/cookies/ - patrz docs/SETUP-WINDOWS.md
#
# WSPIERANE FORMATY:
#
# YouTube (kanaly):
#   https://www.youtube.com/@nazwa
#   https://www.youtube.com/@nazwa/shorts
#   https://www.youtube.com/channel/UCxxxxxxxxx
#
# YouTube (pojedyncze shorty - bez filtra views):
#   https://youtube.com/shorts/XYZ
#
# TikTok (profile):
#   https://www.tiktok.com/@nazwa
#
# TikTok (pojedyncze filmy):
#   https://www.tiktok.com/@nazwa/video/1234567890
#
# Instagram (profile):
#   https://www.instagram.com/nazwa
#   https://www.instagram.com/nazwa/
#
# Instagram (pojedyncze rolki):
#   https://www.instagram.com/reel/ABC123/
#
# REKOMENDACJA:
# - 3-10 kanalow z Twojej niszy mix YT/TT/IG (np. 3 YT + 3 TT + 3 IG)
# - Wymieniaj co 2-3 tygodnie zeby system nie wpadl w 1 styl
# - Zaloz drugie konto TT/IG do podpiecia (cookies) - ochrona glownego konta
